# PRD — 직업 변신 사진관

## 1. 비전
> "내가 의사가 된다면? 우주비행사가 된다면?" — 사진 한 장으로 8가지 직업의 미래 모습을 시각화하는 진로 체험 도구.

## 2. 사용자
- 학교 진로 부스 방문 학생 (초·중·고)
- 진로 교사·운영자

## 3. 사용자 시나리오
1. 사진 업로드 (정면 얼굴)
2. Gemini API 키 입력 (BYOK)
3. 8개 직업 중 1~8개 체크 (기본 1개)
4. 「선택한 N개 직업 변신!」 클릭 → 동시 생성
5. Before/After 슬라이더로 비교
6. 마음에 드는 카드 PNG/ZIP 다운로드 → 인쇄

## 4. 기능 요구사항
- **F1.** 8가지 직업 사전 정의 + 영문 정밀 프롬프트
- **F2.** 1~8개 사용자 선택 (체크박스 + 카운터 + 전체 토글)
- **F3.** 13:10 가로 비율 강제 (프롬프트 + CSS aspect-ratio)
- **F4.** 결과 카드 Before/After 슬라이더 + 더블클릭 전체화면 모달
- **F5.** 개별 PNG / 전체 ZIP 다운로드

## 5. 비기능 요구사항
- BYOK — 키 외부 저장 X
- 단일 `index.html` + Vercel 호스팅
- 8개 동시 호출 시 ~30초 (사용자 안내)
- 429 자동 재시도 2회

## 6. 디자인 원칙
- 부스 메인과 동일한 글래스모피즘 + 오로라 (블루-에메랄드 컬러)
- 인쇄 친화 13:10 카드

## 7. 성공 지표
- 부스 회전율 (1회당 1~3분)
- 학생이 인쇄해 가져가는 비율
- API 비용/체험자 (선택 모드로 평균 1~2장 호출)

## 8. 향후 (Out of scope)
- 직업 데이터 사용자 정의
- 한국 문화 맞춤 옵션 (한복 의사, KARI 우주비행사 등)
- 결과 카드에 직업명/이름 자동 합성 (현재는 그림만)
